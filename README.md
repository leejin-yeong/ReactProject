# SPROUT CHAT

## 장소별 실시간 채팅 페이지 구현

### 사용한 환경 기술
1) React
2) Node.js
3) socket.io
4) Firebase
5) 네이버 API

1) 회원가입, 로그인, 로그아웃
2) 지도 검색, 현재 위치 이동
3) 채팅
